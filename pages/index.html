RuMotoSport Kodi

Free Kodi addon for browsing and streaming MotoGP 2026 race recordings — MotoGP, Moto2, Moto3, and additional sessions.

v1.0.0

Features

🏁 All Championships

MotoGP, Moto2, Moto3, and supplementary sessions — all in one place.

📺 Every Session

FP1, FP2, FP3, Qualifying, Race, Warmup, Press conferences, and more.

▶️ HLS Streaming

Direct HLS stream playback inside Kodi. No external players needed.

📦 Zero Dependencies

Pure Python stdlib — no pip packages or extra libraries required.

📂 Browse by Year

Navigate seasons, championships, race weekends, and individual sessions.

🔄 Auto-Updates

Install via repository for automatic updates when new versions are released.

Navigation

📁 Years
📁 Championships (MotoGP, Moto2, Moto3, Дополнительно)

▶️ Play HLS stream

Installation

Via repository (recommended — auto-updates)

  1. In Kodi: Settings → System → Add-ons → enable Unknown sources
  2. Download repository.rmskodi.zip from the Releases page
  3. Settings → Add-ons → Install from zip file → select repository.rmskodi.zip
  4. Install from repository → RuMotoSport Repository → Video add-ons → RuMotoSport Kodi → Install

Direct ZIP (no auto-updates)

Download plugin.video.rmskodi.zip from the Releases page and install via Add-ons → Install from zip file.
